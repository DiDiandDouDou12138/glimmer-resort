# 微光度假村 · The Glimmer Resort

> 所有进入微光的人，都将成为微光的一部分。

一款**搜索框驱动**的中文民俗恐怖解谜网页游戏。你和女友入住一家过分完美的现代度假酒店，房间里只有一台平板、一个搜索框。观察异样 → 提炼关键词 → 搜索 → 被"劫持"的搜索结果会告诉你这家酒店的真相。

## 怎么玩

- 在线游玩：部署后打开站点即可
- 本地游玩：把 `index.html` 与 `shenshitang.html` 放在同一文件夹，双击 `index.html`
- 卡关时在搜索框输入「提示」：第一次给方向，连续第二次给答案

## 章节

1. **初次入住** —— 46 层的楼，电梯少一个键
2. **云顶盛宴** —— 每桌一盅必须喝完的"安神汤"
3. **镜可照真** —— 镜子里多出来的那扇门，和一个 1998 年的旧网站
4. **亥时·诸镜皆门** —— 回魂夜，先入者为主
5. 终章开发中

## 特性

- 纯前端、零依赖：两个 HTML 文件包含全部剧情、SVG 插画与 WebAudio 音效
- 双窗口解谜：微光管家系统 × 沈氏堂旧门户，线索在那边，答案输回这边
- 六种不重样的"劫持"惊吓效果，看过即不重播（防脱敏）
- 内置线索手账、两级提示、开发者模式（进度条即时跳转任意节点）

## ⚠️ 剧透警告

源码里能看到所有谜底与关键词表。想先玩再看代码。
